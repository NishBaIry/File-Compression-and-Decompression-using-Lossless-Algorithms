# Icons Directory

Place your icon files here for the GUI.

## Required Icons (24x24 or 32x32 PNG recommended)

### Toolbar Icons:
- `add.png` - Add files to archive button
- `extract.png` - Extract files button
- `view.png` - View file button
- `delete.png` - Delete file button
- `info.png` - Archive information button
- `compress.png` - Quick compress button

### Optional Menu Icons:
- `open.png` - Open archive
- `close.png` - Close archive
- `new.png` - New archive
- `exit.png` - Exit application
- `about.png` - About dialog

## Icon Resources

Free icon sources:
- **Icons8**: https://icons8.com (search: archive, folder, extract, compress)
- **Flaticon**: https://www.flaticon.com
- **Material Icons**: https://fonts.google.com/icons
- **Feather Icons**: https://feathericons.com

## Tips:
- Use PNG format with transparency
- Recommended size: 24x24 or 32x32 pixels
- Keep consistent style across all icons
- Use simple, clear designs for better visibility
